Networking Protocols

HTTP

HTTP is a protocol that is used to transfer hypertext documents between a client and a server on a computer network. It is commonly used to transfer web pages from a server to a client.

TCP

TCP is a connection-oriented protocol that is used to transfer data between a client and a server on a computer network. It is commonly used to transfer files and other data that require reliable delivery.

UDP

UDP is commonly used to transfer real-time data, such as audio and video, from a server to a client. The protocol is connectionless and does not guarantee the delivery of data.
